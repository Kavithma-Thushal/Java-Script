jQuery Formalized events

which means mostly using events in JS. to simplify the development process jQuery has developed some methods to access the events in a formalized manner. so they are introduced as formalize events

  • click(function)
  • dblclick(function)
  • foucs(function)
  • blur(function)
  • keydown(function)
  • keyup(function)
  • keypress(function)
  • mousedown(function)
  • mouseup(function)
  • mousemove(function)
